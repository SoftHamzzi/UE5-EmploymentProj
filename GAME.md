# GAME DESIGN - EmploymentProj

## 개요

"취업"을 테마로 한 익스트랙션 슈터.
타르코프의 핵심 루프(스폰-파밍-교전-탈출)에 자판기(뽑기) 시스템을 결합.

- 장르: FPS / Extraction Shooter
- 인원: 2~8인
- 시점: 1인칭
- 매치 시간: 5~10분

---

## 코어 루프

```
매치 시작
  -> 지정된 스폰 위치 중 랜덤 배정
  -> 파밍 (바닥 루팅 + 자판기)
  -> 교전 (플레이어 / AI 적)
  -> 탈출 지점 도달 + 대기
  -> 탈출 성공: 소지품 보존 + 보너스
  -> 사망: 소지품 손실
매치 종료 (제한 시간 초과 시 전원 사망 처리)
```

---

## 매치 흐름

| 상태 | 설명 |
|------|------|
| Waiting | 플레이어 접속 대기, 최소 인원 충족 시 카운트다운 |
| Playing | 매치 진행, 제한 시간 작동 |
| Ended | 시간 초과 또는 전원 탈출/사망, 결과 정산 |

---

## 스폰

- 맵에 스폰 포인트 여러 개 배치
- 매치 시작 시 각 플레이어에게 랜덤 배정
- 플레이어 간 최소 거리 보장 (스폰 킬 방지)

---

## 맵

- 소규모 폐쇄형 구역 (창고, 컨테이너, 건물 밀집)
- 파밍 포인트: 바닥/선반 등에 아이템 스폰
- 자판기: 맵 곳곳에 랜덤 배치
- 탈출 지점: 1~2곳, 맵 가장자리
- 병목 구간 + 열린 공간 혼합으로 교전 유도

---

## 자판기 (핵심 시스템)

파이널스의 금고에서 착안한 고위험-고보상 파밍 시스템.

### 작동 방식
1. 자판기에 접근하여 상호작용
2. 1000원 투입
3. 자판기가 5초간 덜덜 떨리며 **주변에 소리가 퍼짐**
4. 5초 후 아이템 배출

### 설계 의도
- **5초 대기**: 무엇이 나올지 기대감 + 소리로 인한 위치 노출 → 긴장감
- **소리 전파**: 근처 플레이어에게 "누군가 뽑기 중" 알림 → 교전 유도
- **확률적 이득**: 투입 금액 대비 무조건 이득이지만, 등급에 따라 차등

### 배출 아이템

| 등급 | 예시 | 확률 (임시) |
|------|------|-------------|
| 일반 | 탄약, 붕대, 잡템(폐지 등) | 50% |
| 고급 | 회복키트, 좋은 탄약, 가치 아이템 | 30% |
| 희귀 | 무기, 고가 아이템, 퀘스트 아이템 | 15% |
| 전설 | 고급 무기, 최고가 아이템 | 5% |

### 확장 고려
- 현재는 자판기 1종만 구현
- 구조상 자판기 타입별로 다른 아이템 테이블을 가질 수 있도록 설계 (DataAsset)

---

## 전투

### 사격
- 히트스캔 (레이캐스트)
- 반동(Recoil): 연사 시 조준점 상승
- 탄퍼짐(Spread): 이동/점프 시 정확도 감소
- 서버 권한 판정

### 체력
- HP 100
- HP 0 = 사망 → 소지품 드랍

### 무기
- 초기 2종: 권총(Pistol), 소총(Rifle)
- DataAsset 기반

| 속성 | 설명 |
|------|------|
| Damage | 탄당 데미지 |
| FireRate | 연사 속도 |
| Recoil | 반동 크기 |
| Spread | 탄퍼짐 |
| MaxAmmo | 최대 탄약 |
| FireMode | 단발 / 연사 |

### AI 적
- 맵에 소수 배치되는 단순한 적
- 기본 행동: 순찰 → 플레이어 감지 → 추적/사격
- 처치 시 아이템 드랍

---

## 이동

| 동작 | 조작 | 설명 |
|------|------|------|
| 걷기 | WASD | 기본 이동 |
| 달리기 | Shift | 속도 증가, 사격 불가 |
| 앉기 | Ctrl | 속도 감소, 정확도 증가 |
| 점프 | Space | - |
| 에임(ADS) | 우클릭 | 줌인, 이동속도 감소, 정확도 증가 |

---

## 스킬 (GAS)

GAS(Gameplay Ability System)로 구현. 초기 3종.

| 스킬 | 효과 | 쿨타임 |
|------|------|--------|
| Dash | 짧은 거리 순간 이동 | 10초 |
| Heal | 3초 채널링 HP 30 회복 (피격 시 취소) | 20초 |
| Shield | 5초간 피해 50% 감소 | 30초 |

---

## 인벤토리

- 슬롯 기반 (6슬롯)
- 아이템 종류: 무기, 탄약, 회복템, 가치 아이템(판매용), 퀘스트 아이템
- 사망 시 전부 드랍
- 탈출 성공 시 보존

---

## 장비

- 무기 슬롯: 주무기 1 + 보조무기 1
- 장비는 인벤토리에서 장착/해제
- 매치 시작 전 로비에서 장비 세팅 (또는 빈손 진입)

---

## 경제 / 판매

| 항목 | 설명 |
|------|------|
| 탈출 보너스 | 탈출 성공 시 고정 보상 |
| 킬 보상 | 플레이어/AI 처치 시 |
| 아이템 판매 | 탈출 후 가치 아이템/잉여 장비를 판매하여 현금화 |
| 자판기 비용 | 1회 1000원 |

- 벌어들인 돈으로 자판기 이용, 장비 구매 등에 사용

---

## 퀘스트 (취업 시스템)

게임의 진행 목표. "취업"이라는 테마로 포장.

### 기본 구조
- 특정 퀘스트 아이템들을 매치에서 수집 (자판기, 파밍, 적 처치 등으로 획득)
- 필요한 퀘스트 아이템을 모두 모으면 "취업 성공" → 회사 맵 해금

### 초기 구현 범위
- 퀘스트 아이템 수집 목록 (예: 이력서, 자격증, 포트폴리오 USB 등)
- 수집 진행도 UI
- 전부 모으면 "취업 완료" 표시

---

## 초기 구현에서 제외하는 항목

아키텍처 설계 시 확장을 막지 않되, 당장 구현하지 않는 요소들.

| 항목 | 설명 | 제외 사유 |
|------|------|-----------|
| 대형 물리 아이템 | TV, 컴퓨터 본체 등 물리적으로 운반해야 하는 아이템 | 물리 운반 + 차량 시스템 필요, 코어와 무관 |
| 차량 | 대형 아이템을 싣고 탈출하는 용도 | 대형 아이템과 세트, 후순위 |
| 자판기 다종화 | 등급별/테마별 자판기 여러 종류 | 1종으로 시스템 증명 충분. DataAsset 구조로 확장 가능하게만 설계 |
| 회사 맵 | 취업 후 진입하는 상위 맵. 더 좋은 뽑기 + 강한 적 | 퀘스트 수집까지만 초기 구현. 회사 맵은 콘텐츠 확장 단계 |
| 실적/퇴직 시스템 | 회사 맵 진입 후 7일 내 실적 미달 시 퇴직 | 회사 맵과 세트 |
| 회사 선택 분기 | 여러 회사 중 택 1, 선택 시 다른 회사 잠금 | 회사 맵과 세트 |
| 팀 모드 | 듀오/스쿼드 | GameMode 확장으로 가능, 솔로 먼저 |
| 로비/매칭 UI | 매치메이킹, 장비 세팅 화면 | 코어 게임플레이 이후 |
| 캐릭터 커스텀 | 스킨, 외형 | 콘텐츠 확장 단계 |
