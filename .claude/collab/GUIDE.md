# 협업 문서 가이드

## 구조

```
.claude/collab/
├── GUIDE.md          ← 이 문서 (사용법)
├── STATUS.md         ← 전체 진행 상황 (메인 에이전트 관리)
├── core.md           ← Core Framework 에이전트 작업 문서
└── replication.md    ← Replication 에이전트 작업 문서
```

## 규칙

### 메인 에이전트
1. STATUS.md 관리 (전체 진행 상황, 에이전트 상태)
2. 서브 에이전트에게 작업 배분
3. 서브 에이전트 작업 문서 관찰 및 통제
4. 작업 완료 시 .claude/results/에 결과 문서 작성

### 서브 에이전트
1. 작업 시작 전: 자기 담당 문서 읽기
2. 작업 중: 진행 상황 업데이트
3. 작업 완료: 완료 섹션에 기록, STATUS.md 상태 변경
4. 이슈 발생: STATUS.md 이슈/블로커 섹션에 기록

### 문서 업데이트 형식
```markdown
## 진행 중 작업
### [작업명]
- [x] 완료한 항목
- [ ] 남은 항목
- 메모: ...
```

## 결과 문서

작업 완료 시 `.claude/results/`에 생성:

```markdown
# [작업명] 결과

## 변경된 파일
- path/to/file.h (추가/수정/삭제)

## 구현 요약
- ...

## 수동 작업 필요 (사용자/MCP)
- [ ] 블루프린트에서 XX 설정
- [ ] 에디터에서 XX 확인
```
